##  聚合查询
select  * from emp;
select  *  from dep;

两个表合在一起查询

- 外连接之右连接
	- 以右表为主表，根据右表数据匹配左表，右表数据是全的，而左表若匹配不上则null
```sql
# emp.dep_id == dep.id

# 查询lala员工的部门名称是什么？
# 将emp和dep进行数据合并，根据指定条件进行数据合并

select dep.name from emp inner join dep on emp.dep_id = dep.id

where emp.name = 'lala';
