


> 数据库其他概念

下面思考一个问题：数据库是如何存储数据的呢？那我们必须先熟悉如下几个概念。

- 数据： data
	>描述事物有无之间的符号记录称为数据，描述事物的符号既可以是数字，也可以是文字，图片，图像，声音，视频，语言等，数据由多种表现形式，它们都可以经过数字化后存入计算机。

- 例如：描述一个人的基本信息就是一组数据，在数据库中叫一行记录
```sql
name age sex
boxbo 18 男
```
- 表：table
>table即文件，用来存放多行内容/多条记录
- 数据库： database(DB)
>即存放数据的仓库，我们可以把它想象成文件夹，里面存在
